## 项目介绍
基于ssm的学生成绩管理系统：前端 jsp、jquery、bootstrap，后端 springmvc、spring、mybatis，角色分为管理员；集成学生信息、课程信息、成绩信息等功能于一体的系统。

## 功能介绍

- 学生信息：学生信息的增删改查，根据学号搜索查询，批量删除
- 课程信息：课程信息的增删改查，根据课程号搜索查询，批量删除
- 成绩信息：成绩信息的增删改查，根据学号搜索成绩，批量删除
